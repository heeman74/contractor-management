    1. Create note_dao.dart (follow BookingDao pattern):
       - class NoteDao extends DatabaseAccessor<AppDatabase> with _$NoteDaoMixin
       - insertNote(companyId, jobId, authorId, body) -> String (returns note id): Transaction with insert to jobNotes + sync_queue entry (entityType: 'job_note', operation: 'CREATE')
       - watchNotesForJob(jobId) -> Stream<List<JobNote>>: Watch jobNotes where jobId matches AND deletedAt is null, ordered by createdAt desc (newest first)
       - upsertFromSync(Map<String, dynamic> data) -> void: Insert or update from sync pull

    2. Create attachment_dao.dart:
       - class AttachmentDao extends DatabaseAccessor<AppDatabase> with _$AttachmentDaoMixin
       - insertAttachment(companyId, noteId, attachmentType, localPath, thumbnailPath, caption, sortOrder) -> String: Insert with uploadStatus='pending_upload'. NO sync queue entry (attachments use separate upload service).
       - watchAttachmentsForNote(noteId) -> Stream<List<Attachment>>: Watch attachments for a note, ordered by sortOrder
       - getPendingUploads() -> Future<List<Attachment>>: Select where uploadStatus='pending_upload' OR uploadStatus='failed', deletedAt is null
       - setUploadStatus(id, status) -> void: Update uploadStatus
       - markUploaded(id, remoteUrl) -> void: Set uploadStatus='uploaded', remoteUrl=url
       - incrementRetry(id) -> void: Set uploadStatus='failed' (AttachmentUploadService handles retry count externally)
       - upsertFromSync(Map<String, dynamic> data) -> void: Insert or update from sync pull (sets remoteUrl from server)

    3. Create time_entry_dao.dart:
       - class TimeEntryDao extends DatabaseAccessor<AppDatabase> with _$TimeEntryDaoMixin
       - clockIn(companyId, jobId, contractorId) -> String: Transaction — first query for any active session (clockedOutAt is null) for this contractorId, if found auto-clock-out (compute duration, set status=completed, enqueue UPDATE to sync_queue), then insert new entry with clockedInAt=now, enqueue CREATE to sync_queue. Returns new entry id.
       - clockOut(entryId) -> void: Transaction — update clockedOutAt=now, compute durationSeconds, set sessionStatus='completed', enqueue UPDATE to sync_queue.
       - watchActiveSession(contractorId) -> Stream<TimeEntry?>: Watch for entry where contractorId matches AND clockedOutAt is null (at most one row)
       - watchEntriesForJob(jobId) -> Stream<List<TimeEntry>>: Watch time_entries for job, ordered by clockedInAt desc
       - upsertFromSync(Map<String, dynamic> data) -> void

    4. Add GPS update method to existing job_dao.dart:
