       - On tap flow:
         a. Check if location services enabled: Geolocator.isLocationServiceEnabled(). If not, show snackbar "Enable location services in Settings".

         c. If deniedForever: show dialog explaining why GPS is needed + "Open Settings" button that calls Geolocator.openAppSettings(). Address field remains manually editable.
         d. If existing address is not null: show confirm dialog "Replace existing address with GPS location?" with Cancel/Replace buttons. If cancelled, return.
         e. Show loading state on button ("Getting location..." with CircularProgressIndicator)
         f. Get position: Geolocator.getCurrentPosition(locationSettings: LocationSettings(accuracy: LocationAccuracy.high))
         g. Store raw lat/lng: call JobDao.updateJobGps(jobId, position.latitude, position.longitude) via GetIt
